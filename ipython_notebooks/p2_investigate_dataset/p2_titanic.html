Introduction¶

In this project I will analyze passenger data from the passenger liner RMS Titanic's tragic maiden voyage in April of 1912. I will use the dataset located at https://www.udacity.com/api/nodes/5420148578/supplemental_media/titanic-datacsv/download?_ga=1.248296128.651510525.1453551417 as the basis for my analysis. The dataset "contains demographics and passenger information from 891 of the 2224 passengers and crew on board the Titanic".

Dataset description from Kaggle:

VARIABLE DESCRIPTIONS:
survival        Survival
                (0 = No; 1 = Yes)
pclass          Passenger Class
                (1 = 1st; 2 = 2nd; 3 = 3rd)
name            Name
sex             Sex
age             Age
sibsp           Number of Siblings/Spouses Aboard
parch           Number of Parents/Children Aboard
ticket          Ticket Number
fare            Passenger Fare
cabin           Cabin
embarked        Port of Embarkation
                (C = Cherbourg; Q = Queenstown; S = Southampton)

I will start by doing some exploratory data analysis with the following initial question in mind:

  • Which factor(s) had the greatest impact on passenger survival rate?

#One-way Anova test for statistical significance
from scipy import stats

embarked_cherbourg = titanic[titanic['Embarked'] == 'C']
embarked_queenstown = titanic[titanic['Embarked'] == 'Q']
embarked_southampton = titanic[titanic['Embarked'] == 'S']

f,p = stats.f_oneway(
    embarked_cherbourg['Survived'],embarked_queenstown['Survived'],embarked_southampton['Survived'])

print("F-statistic: {0}    P: {1}".format(round(f,2),p))

The analysis of the Titanic dataset suggests that three major factors affecting the survival rate of passengers on the Titanic were:

  1. sex (females were 25% more likely to survive the incident, regardless of other factors such as class and age)
  2. class (first class passengers were 133% more likely to survive than passengers in 2nd class, and 260% more likely to survive than passengers in 3rd class)
  3. and finally, age, where children between the ages of 0 and 5 years were 182% more likely to survive than all other passengers.

A limitation of the dataset is that it doesn't contain all titanic passengers, and none of its crew members. It is unclear from the description of the dataset how the data was collected, including whether there might have been any particular biases in the sampling process. Although the overall survival percentage of the sample (38.4%) and all passengers (37.7%) are very similar, there still might be relevant nuances in the missing passenger data altering the survival rate of the passenger groups. Another limitation is the inconsistent quality of the passenger records in the dataset, where 73.2% of the first class passenger records are complete, while only 8.2% of 2nd class passengers and 2% of 3rd class passengers are complete. It is likely that more 3rd class passengers were excluded from the dataset due to missing data than 1st class passengers with missing data.

Although it's outside the scope of this project report, it would be interesting to take a look at some of the other factors potentially affecting survival rate. For example, were people with the same last name more likely to survive together, indicating traveling with family being a relevant factor for survival? Did people traveling alone have a greater or worse chance of surviving? If we had access to crew data, it could be interesting to see if working close to the 1st class cabins increased the chances of survival for the crew, and if the crew survival rate is similar to that of the 1st class passengers.
